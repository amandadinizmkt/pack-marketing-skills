## 🎯 O que é isso?

**Skills** são instruções especializadas que ensinam o Claude a dominar tarefas específicas. É como contratar um especialista sob demanda — mas de graça e disponível 24/7.

Este pack reúne **445+ skills focadas 100% em marketing**, organizadas em 12 categorias que cobrem toda a jornada do profissional de marketing digital:

```
📝 Da copy que converte...
📧 ...ao email que nutre...
🔍 ...ao SEO que rankeia...
📢 ...ao anúncio que escala...
📊 ...ao dashboard que comprova ROI.
```

> 💡 **Feito por marketeiros, para marketeiros.** Sem código, sem terminal, sem complicação.

## 📖 Como usar

### No Claude.ai (mais fácil)

1. Acesse [claude.ai](https://claude.ai)
2. Vá em **Configurações** → **Habilidades** → **+**
3. Faça upload do arquivo `SKILL.md` da skill desejada
4. Pronto! O Claude agora é especialista naquele assunto ✨

### No Claude Code (terminal)

```bash
# Instalar uma skill específica
claude skill add caminho/para/skill/

# Ou clonar o pack inteiro
git clone https://github.com/amandadinizmkt/pack-marketing-skills.git
```

> 🔥 **Dica:** você pode adicionar **várias skills ao mesmo tempo**. Elas ficam disponíveis em todos os seus chats.

### 🔗 A skill mais importante: `product-marketing-context`

Antes de usar qualquer outra skill, configure esta. Ela cria um **documento de posicionamento** com:
- Quem é seu público (ICP)
- Sua proposta de valor
- Tom de voz da marca
- Diferenciais competitivos

> Todas as outras skills leem esse contexto automaticamente, gerando resultados **muito mais relevantes** para o seu negócio.

## ❓ FAQ

<details>
<summary><strong>🆓 Preciso pagar para usar?</strong></summary>
<br/>
Não! As skills são 100% gratuitas. Você só precisa de uma conta no <a href="https://claude.ai">claude.ai</a> (tem plano gratuito).
</details>

<details>
<summary><strong>🔄 Posso usar várias skills ao mesmo tempo?</strong></summary>
<br/>
Sim! Adicione quantas quiser — elas ficam disponíveis em todos os seus chats. Recomendamos começar com <code>product-marketing-context</code> + 2-3 skills da sua área principal.
</details>

<details>
<summary><strong>🤷 Preciso saber programar?</strong></summary>
<br/>
Não. Zero código necessário. Se você sabe usar o ChatGPT ou o Claude, já sabe usar as skills. É só fazer upload do arquivo.
</details>

<details>
<summary><strong>🇧🇷 As skills funcionam em português?</strong></summary>
<br/>
Sim! Todas as skills foram pensadas para o mercado brasileiro. O Claude responde no idioma que você escrever.
</details>

<details>
